# Vim 1337 Config

Simple Vim configuration adapted to the requirements of 1337 school (norm, C, minimal comfort).

---

## Installation
```bash
git clone https://github.com/Choubi-Mohammed/myconfig-vim-13.git
cd myconfig-vim-13
chmod +x setup_vim_13.sh
./setup_vim_13.sh
```

---

## Features
- Indentation compliant with the 1337 norm (4 spaces)
- Limit column at 80 characters
- Display of invisible characters
- C compilation shortcut (F5)
- Customizable theme

---

## Theme Preview
```bash
chmod +x preview_vim_colors.sh
./preview_vim_colors.sh
```

---

## Configured File
The script creates/modifies:
```bash
~/.vimrc
```
A backup is automatically made if the file already exists.
